# IPL_captstone-for-Visualisation
This capstone project focuses on exploratory data analysis (EDA) and visualization of the Indian Premier League (IPL) dataset sourced from Kaggle.
Using Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn, the project answers meaningful cricket-related questions through data-driven insights and visual storytelling.

The goal of this project is to demonstrate strong data analysis, visualization, and problem-solving skills using real-world sports data.

Technologies & Libraries Used

Python

NumPy – Numerical computations

Pandas – Data cleaning & manipulation

Matplotlib – Data visualization

Seaborn – Statistical and advanced visualizations
